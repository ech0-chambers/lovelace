LoveLace

Information

This app can convert instructions for hand-knitted lace into a chart, punchcard, or image file for AYAB. Since this is intended for use with a knitting machine, all rows must have the same number of stitches, and purl stitches are not recognised. To load an example (taken from A Second Treasury of Knitting Patterns) click here:

The input is quite flexible, but design specifically with the syntax used in A Second Treasury of Knitting Patterns in mind. Full words (e.g. yarn over) and abbreviations (yo) are accepted. Below is a list of acceptable instructions.

  • Increases:
    • Yarn Over, yo
  • Decreases:
    • Slip-slip-knit, ssk
    • Knit 2 together, k2 tog
  • Double Decreases:
    • Slip 1 - knit 2 together - pass slipped stich over, sl 1 - k2 tog - psso
    • Slip 2 - knit 1 - pass two slipped stitches over, sl 2 - k1 - p2sso
    • Knit 3 together, k3 tog
  • Others:
    • Knit stitches, k[number], for example knit 6 or k6
    • To repeat a set of instructions, use ( ... ) twice, or ( ... ) four times etc

Information

This app can convert instructions for hand-knitted lace into a chart, punchcard, or image file for AYAB. Since this is intended for use with a knitting machine, all rows must have the same number of stitches, and purl stitches are not recognised. To load an example (taken from A Second Treasury of Knitting Patterns) click here:

The input accepts the following symbols:

Symbol Meaning
  (space), ., - Knit
/ Knit 2 together (right-leaning decrease)
\ Slip slip knit (left-leaning decrease)
^, x, X Slip 2, knit 1, pass slipped stitches over (central double decrease)
o, O, 0 Yarn over (increase)

                                    all double decreases ("^") are interpreted as a sl 2 - k1 - p2sso, since this is the
                                    most reliable double decrease on a knitting machine. If you need a specific double
                                    decrease, use the instruction input instead.
